Thank you for using EpiData.

If You installed a previous version of EpiData, then
a. cancel this installation now 
b. un-install the previous version (see control-panel)
c. rerun this installation program.


If You are new to EpiData then we
suggest that You start working with 
EpiData by reading the file 
ReadMe.Rtf which is shown when you 
execute EpiData the first time.

Read EpiTour (found in the help
menu) to get a first impression
of what EpiData can do and how
it is done.
